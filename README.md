# 🎨 Planner_Blender_3D

**Автоматический учёт времени для проектов в Blender 3D**

![Version](https://img.shields.io/badge/version-1.0.1-orange)
![Platform](https://img.shields.io/badge/platform-Windows-blue)

*  ***✨ Основные возможности***
*  ***🚀 Удобный запуск:*** Открывайте проекты в Blender прямо из программы. Никаких поисков файлов в папках.
*  ***🤖 Автоматический таймер:*** Запускается сам при открытии **.blend файла**. Закрыли Blender → остановился.
*  ***⏱️ У каждого проекта — свой таймер:*** Время отслеживается отдельно. Никакой путаницы.
*  ***📊 Подробные отчёты:*** Дневная статистика по каждому проекту. Сколько работали сегодня, вчера, на прошлой неделе.
*  ***📅 Визуальный контроль дедлайнов:*** Проекты подсвечиваются жёлтым (осталось 2 дня) и красным (менее суток).
*  ***🎨 5 тем оформления:*** От классического оранжевого Blender до «фиолетовой ночи».
*  ***✉️ Обратная связь:*** Прямо из программы — напишите идею или сообщите об ошибке.

**📥 Установка**
Скачайте Planner_Blender_3D_Setup.exe ниже
Запустите файл
Следуйте инструкциям установщика
При первом запуске выберите папку для хранения данных
**⚠️ Важно**
Антивирус может ошибочно заблокировать программу (нет цифровой подписи). Добавьте файл в исключения или временно отключите антивирус при установке.
