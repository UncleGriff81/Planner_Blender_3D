# 🎨 Planner_Blender_3D

**Автоматический учёт времени для проектов в Blender 3D**

![Version](https://img.shields.io/badge/version-1.0.1-orange)
![Platform](https://img.shields.io/badge/platform-Windows-blue)

## 🎯 Основные возможности

| Функция | Описание |
|---------|----------|
| ⏱️ **Автоматический таймер** | Запускается сам при открытии `.blend` файла |
| 📅 **Дедлайны** | Визуальная индикация: жёлтый → 2 дня, красный → менее суток |
| 🎨 **5 тем оформления** | От оранжевой (в стиле Blender) до тёмной |
| 📊 **Отчёты** | Дневная статистика по каждому проекту |
| ✉️ **Обратная связь** | Прямо из программы |

## 📥 Установка

1. Скачай `Planner_Blender_3D.exe` из раздела **Releases**
2. Запусти файл
3. При первом запуске выбери папку для данных

## ⚠️ Важно

Антивирус может ошибочно заблокировать программу. Добавь файл в исключения.
