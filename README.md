**Planner_Blender_3D** — автоматический учёт времени для Blender 3D.
## О программе
- ⏱️ Таймер стартует сам при открытии .blend
- 📅 Дедлайны с цветовой индикацией
- 🎨 5 тем оформления
- 💾 Автосохранение каждые 30 секунд
- ✉️ Отправка идей разработчику
- 🆓 Бесплатно, без установки
- 🖥️ Windows / Linux / macOS

> Студенческий проект. Создан с помощью нейросети DeepSeek.
