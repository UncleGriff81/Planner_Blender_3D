# Planner_Blender_3D
Planner_Blender_3D — программа для автоматического учёта времени в проектах Blender 3D. Таймер запускается сам при открытии .blend файла. Дедлайны, 5 тем оформления, автосохранение, отправка идей разработчику. Студенческий проект. Бесплатно. Не требует установки. Поддерживает Windows, Linux, macOS.
